# History Timeline Challenge Game

<img src="{{ '/images/projects/history-timeline-challenge.png' | url }}" alt="History Timeline Challenge Game" class="post-featured-image" />

I decided to create this history game as my first attempt at using Claude Code to create an interactive. Previously, I might have used H5P, Genially, or Oppia to create some sort of branching scenario for this activity, but I decided to test the capabilities of Claude Code. I managed to create the entire game in just a few hours. I played around a bit with the leaderboard and would be excited to see what more I could do with that to make it a true leaderboard (one option would be to use a Google form, while another could involve creating a SCORM package to run this in an LMS).
