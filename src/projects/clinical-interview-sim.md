# Clinical Interview Simulation

<img src="{{ '/images/projects/clinical-interview-sim.png' | url }}" alt="Clinical Interview Simulation" class="post-featured-image" />

<div class="problem-solution">
  <div class="problem-solution-col">
    <div class="section-eyebrow">The Problem</div>
    <p>Psychology students rarely get low-stakes opportunities to practice the diagnostic interview process. They learn about the process in theory, but lack practical applications and examples of how they might approach the interview.</p>
  </div>
  <div class="problem-solution-col">
    <div class="section-eyebrow">The Solution</div>
    <p>An AI-powered simulation where students interview a virtual patient and practice the diagnostic process, built with Claude Code.</p>
  </div>
</div>

I've been meeting with Joe Slade, a psychologist and AI expert from Oregon State University, about ways to help students get more comfortable using AI for practical use cases outside of just as a "cheating-helper" or "quiz-taker." Joe has some phenomenal ideas that have sparked a lot of great conversations. One idea he had was to create a chatbot to respond as a patient in order to give students practice with the diagnostic process. I thought it'd be fun to mock this up in a more structured way, so I played with Claude Code to create this diagnostic activity.

[Try the Clinical Interview Simulation](https://wendyhollandking.github.io/clinical-interview-sim/)
